# ContentFlow AI

## Idea Summary
A SaaS platform that automatically generates 30-day content calendars for Instagram and LinkedIn, specifically targeting small business owners in India.

## Problem Statement
Small business owners in India struggle to consistently create and manage 30-day content calendars for their Instagram and LinkedIn presence, leading to inconsistent social media engagement and marketing efforts.

## Target Persona
Indian small business owner

## MVP Features
- User onboarding and business profile setup (niche, target audience, goals)
- Automated 30-day content calendar generation for Instagram and LinkedIn
- Daily content ideas with suggested captions and relevant hashtags
- Ability to review, edit, and customize generated content
- Calendar view to manage and visualize planned posts
- Basic user dashboard with subscription management
- Export content calendar (e.g., CSV, PDF) for manual scheduling

## Tech Stack
Python (for ML/backend logic), a modern web framework (e.g., Django/Flask or Node.js/Express), React/Vue for frontend, PostgreSQL for database, cloud hosting (AWS/GCP/Azure) for scalability, and integration with LLM APIs (e.g., OpenAI, Hugging Face).
